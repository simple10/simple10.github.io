CSS Only + Accessible Input Toggle Switch

CSS input toggle switches are nothing new, but I’ve yet to see one that is Bootstrap 3 compatible, doesn’t use JavaScript, accepts tab focus and styles properly, and doesn’t break accessibility. So I wrote one.

Demo on JSFiddle - but without JS ;-)

If you have stacked toggles, just add toggle-align class to <div class="toggle toggle-align"> to align text. This is optional, but looks better IMHO.

Other toggle switches for reference:

  • http://www.bootstrap-switch.org/
  • http://minhur.github.io/bootstrap-toggle/
  • http://ghinda.net/css-toggle-switch/
